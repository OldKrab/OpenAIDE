# Code of conduct

OpenAIDE contributors must keep project spaces constructive, professional, and
safe. Be respectful, discuss technical decisions in good faith, and focus
criticism on work rather than people. Harassment, threats, discriminatory conduct,
doxxing, and deliberate disruption are not acceptable.

Maintainers may edit or remove contributions and restrict participation when
needed to protect the community. Report ordinary moderation concerns to the
repository maintainers. For sensitive reports, use the repository's private
security-advisory channel described in `SECURITY.md`.
